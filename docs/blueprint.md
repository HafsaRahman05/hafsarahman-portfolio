# **App Name**: Portfolio Pro

## Core Features:

- Hero Section: Display name, role (Software Engineering Student), tagline focused on AI, ML, and Web Development, and buttons linking to projects, GitHub, and LinkedIn.
- About Me Section: Introduce professional background and highlight interest in AI, ML, Databases, and Web & Game Development.
- Skills Showcase: Categorize and present programming, AI/ML, database, and tool skills.
- Project Cards: Display AI, ML, Database, and Web projects as cards, including project title, description, tech stack, live link, and GitHub repository link. LLM tool chooses card layout.
- Category-based Project Filtering: Allow filtering of projects by category (AI/ML/Database/Web).
- Contact Form: Professional contact form (Name, Email, Message) with data stored in Firestore.
- Social Links: Link to GitHub and LinkedIn profiles using social icons.

## Style Guidelines:

- Primary color: Deep blue (#30475E) to convey professionalism and trustworthiness.
- Background color: Light gray (#F0F0F0) for a clean, modern feel.
- Accent color: Teal (#26A69A) to add a touch of modernity and highlight key elements.
- Body font: 'Inter' sans-serif, for a modern, neutral look; suitable for headlines or body text
- Code font: 'Source Code Pro' for displaying code snippets.
- Use a set of consistent and professional icons. Consider line icons for a modern aesthetic.
- Implement a clean, minimal layout with a focus on readability and user experience. Use a grid-based system for responsiveness.
- Add subtle animations and hover effects to enhance user interaction without being distracting.
